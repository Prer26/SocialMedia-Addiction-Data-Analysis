## Overview
Interactive Power BI dashboard analyzing social media addiction and its impact on sleep, mental health, relationships, and academic performance.
Built through a friendly Power BI tug of war, focusing on visual design, interactivity, and data storytelling.

## Project Objectives
* Understand social media addiction patterns across countries and age groups
* Analyze the relationship between addiction, sleep hours, and mental health
* Observe academic impact using interactive visualizations

## Dashboard Features
Interactive slicers for:
*Country selection (e.g., Afghanistan)
*Age selection (e.g., 23)
*Dynamic KPIs that update in real time
*Country-wise academic impact analysis
*Gender-wise addiction comparison
*Age vs sleep hours visualization

## KPIs Tracked

* Average Social Media Addiction Score
* Average Sleep Hours
* Academic Impact
* Relationship Status Distribution

## Key Insights

* Higher social media addiction correlates with reduced sleep hours
* Academic impact varies significantly across countries
* Young adults show higher addiction levels compared to other age groups
